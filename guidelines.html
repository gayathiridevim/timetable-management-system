System Guidelines

Instructions for using the timetable management system

Steps to Use the System

  1. Department Setup
    Create departments and define the number of sections for each department.
  2. Subject Management
    Add all subjects for the department and specify whether the subject is Theory or Lab.
  3. Staff Management
    Add faculty members, assign subjects, sections, and set the maximum workload (periods per week).
  4. Section Management
    Manage the available sections for each department.
  5. Generate Timetable
    Automatically generate the timetable for each section based on subjects and available periods.
  6. View Timetable
    View the generated timetable, edit if required, and download the timetable as a PDF.
  7. Staff Allocation
    The system displays which staff member is assigned to each subject for the selected section.

Important Notes

  • Each subject can be assigned to a staff member.
  • Different sections can have different staff for the same subject.
  • The timetable can be edited before downloading.
  • The PDF download includes staff allocation details.
  • Ensure all subjects and staff are added before generating the timetable.
